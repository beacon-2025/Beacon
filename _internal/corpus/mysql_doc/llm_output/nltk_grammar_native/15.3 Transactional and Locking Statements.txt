```
S -> TRANSACTION_STATEMENT

TRANSACTION_STATEMENT -> LOCAL_TRANSACTION | XA_TRANSACTION

LOCAL_TRANSACTION -> SET_AUTOCOMMIT | START_TRANSACTION | COMMIT | ROLLBACK
SET_AUTOCOMMIT -> 'SET' 'autocommit'
START_TRANSACTION -> 'START' 'TRANSACTION'
COMMIT -> 'COMMIT'
ROLLBACK -> 'ROLLBACK'

XA_TRANSACTION -> 'XA' XA_STATEMENT
XA_STATEMENT -> 'transaction_statement_unknown'
```
